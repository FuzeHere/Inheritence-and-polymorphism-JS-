# JavaScript OOP Inheritance Example 🐇🦅

Project ini adalah contoh sederhana **Object-Oriented Programming (OOP)** di JavaScript yang menunjukkan konsep **Inheritance (Pewarisan)**.

Pada kode ini terdapat:

- **Parent Class**: `Animal`
- **Child Class / Subclass**: `Rabbit` dan `Eagle`

Tujuan project ini adalah untuk memahami bagaimana subclass dapat mewarisi properti dari parent class, lalu memiliki method khusus masing-masing.

---

## 📌 Konsep yang Dipelajari

Project ini membahas beberapa konsep dasar OOP:

- **Class**
- **Constructor**
- **Inheritance (Pewarisan)**
- **extends**
- **super()**
- **Method pada subclass**

## 📖 Ringkasan Singkat

**Animal = class induk**

**Rabbit = turunan dari Animal, punya method eat()**

**Eagle = turunan dari Animal, punya method fly()**

**extends = untuk pewarisan**

super() = untuk memanggil constructor parent
